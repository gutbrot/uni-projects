# Leírás
Kettő, koncepciójában igen hasonló, szerepjáték (dungeon crawler). Alapvetően körökreosztott játékmenet, amelyben a pályán diagonálisan lehet mozogni, szörnyeket legyőzni és az ezzel nyert XP-vel fejődni és új képességekre szert tenni.

## Text-based
**BME Programozás alapjai 2 tárgy kereteiben készült**
[Egyetemi oldal](https://infocpp.iit.bme.hu/nhf)
- CPP-ben készült
- karakteres pályamegjelenítés, billentyű parancs alapú irányítás
- attribútum alapú képességrendszer
  - amennyiben bizonyos attribútumok fejleszt a játékos, úgy válnak elérhetővé további képességek   

## Swing based
**BME Programozás alapjai 3 tárgy kereteiben készült**
[Egyetemi oldal](https://www.iit.bme.hu/oktatas/tanszeki_targyak/BMEVIIIAB00)
- Java, Swing GUI-val
- több panel a grafikus megjelenítéshez, kattintás alapú irányítás
- gráf alapú képességrendszer
  - az XP-t különböző képességekre lehet elkölteni, amelyik újabb képességeket nyitnak meg
